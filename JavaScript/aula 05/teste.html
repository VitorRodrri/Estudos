O que eu aprendi

Eu aprendi que para comerçar precisamos do html, o css, e javascript. O html define o significado e a estrutura do contúdo web, o css é usado para a aparencia e apresentação e o javascript é usado para interação, funcionalidade e comportamento

Comandos

Se quiser mudar só o titulo ou o paragrafo tem que criar uma linha só com o comando do mesmo. ex: style p1 ou h1 etc...
